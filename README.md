Gyazo CLI
============================================================

Gyazo CLI Client

Require
------------------------------------------------------------

- curl

Settings
------------------------------------------------------------

set enveropments `GYAZO_PATH` to Gyazo server url

```shell
 $export GYAZO_PATH=http://gyazo.com/
```


Usage
------------------------------------------------------------

```shell
 $gyazo /path/to/push/image/file.jpeg
```
